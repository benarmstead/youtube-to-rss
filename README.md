# Installation

`git clone https://github.com/benarmstead/youtube-to-rss`

`cd youtube-to-rss`

`python3 main.py`

## Dependancies
GTK (tested on GTK3)



Takes a youtube channel id and finds the RSS feed for the channel as can be seen below.
